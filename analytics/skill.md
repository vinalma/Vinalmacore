# Skill: Analytics

## Quando usar
- Interpretar dados de campanha, site ou funil

## Inputs mínimos
- Plataforma (Meta, GA4, etc.)
- Período analisado
- Objetivo (lead, venda, tráfego)

## Métricas por objetivo

### Tráfego / Geração de lead
| Métrica | Meta saudável |
|---|---|
| CTR (link) | > 1,5% |
| CPL | Depende do ticket |
| Frequência | < 3,5 por semana |
| Taxa de conversão LP | > 20% (captura) |

### Vendas
| Métrica | Meta saudável |
|---|---|
| ROAS | > 3x |
| CPA | < 30% do ticket |
| LTV/CAC | > 3x |

## Diagnóstico rápido

```
CTR baixo → problema no criativo ou público
CPL alto + CTR ok → problema na landing page
Taxa de conversão baixa → problema na oferta ou copy
ROAS baixo + conversão ok → problema no ticket ou LTV
```
