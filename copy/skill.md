# Skill: Copy

## Quando usar
- Criar headlines, CTAs, copy de anúncio, email, landing page

## Inputs mínimos
- Produto/serviço
- Público-alvo
- Objetivo (cliques, leads, vendas)
- Tom (direto, emocional, técnico)

## Estrutura padrão

### Headline
[Problema ou desejo] → [Resultado] → [Diferenciador]

### Body
- Dor
- Agitação
- Solução
- Prova
- CTA

## Template anúncio direto

```
[HOOK — 1 linha, corta o scroll]

[Dor ou desejo em 1 frase]

[Solução + diferenciador]

[Prova social curta]

[CTA claro]
```

## Template email curto

```
Assunto: [Curiosidade ou benefício direto]

[Abertura pessoal — 1 linha]

[Problema do leitor]

[Solução que você oferece]

[CTA único]
```
