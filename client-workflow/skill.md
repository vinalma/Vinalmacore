# Skill: Fluxo de Cliente

## Quando usar
- Onboarding, gestão de entregáveis, comunicação e encerramento

## Etapas padrão

```
1. Briefing       → Coletar tudo antes de começar
2. Planejamento   → Alinhar expectativas e prazo
3. Execução       → Entregar em ciclos curtos
4. Aprovação      → Revisão clara e objetiva
5. Go-live        → Publicar / ativar
6. Acompanhamento → Primeiros resultados
7. Relatório      → Resultado + próximos passos
```

## Briefing mínimo

```
Cliente:
Produto/serviço:
Público-alvo:
Objetivo da campanha:
Ticket médio:
Orçamento de tráfego:
Prazo:
Canais disponíveis:
Materiais existentes (site, copy, criativos):
Resultado esperado:
```

## Comunicação

- Update semanal obrigatório (mesmo sem resultados novos)
- Resposta em até 24h em dias úteis
- Decisões importantes por escrito

## Encerramento

```
1. Relatório final de resultado
2. Entrega de acessos e arquivos
3. Depoimento / case (solicitar)
4. Offboarding limpo
```
