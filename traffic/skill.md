# Skill: Tráfego

## Quando usar
- Criar campanhas, segmentar audiências, revisar criativos para anúncios

## Inputs mínimos
- Plataforma (Meta, Google, TikTok)
- Objetivo (lead, tráfego, conversão)
- Orçamento diário
- Público (demo, interesses ou persona)

## Checklist pré-campanha

- [ ] Pixel/Tag instalado e testado
- [ ] Evento de conversão configurado
- [ ] Criativo aprovado (thumb + hook + CTA)
- [ ] Landing page funcional e rápida
- [ ] UTMs configurados

## Estrutura padrão Meta Ads

```
Campanha → Objetivo
  └─ Conjunto → Audiência + Orçamento
        └─ Anúncio → Criativo + Copy + CTA
```

## Segmentação rápida

| Tipo | Quando usar |
|---|---|
| Interesse | Audiência fria, topo de funil |
| Lookalike | Escalar o que já funciona |
| Remarketing | Quem visitou / engajou |
| Broad | Campanha com budget alto, deixa o algoritmo |

## Otimização

- Aguarde 3–5 dias antes de otimizar
- Corte anúncios com CPL > 2x da meta
- Teste 1 variável por vez (thumb, hook, público)
