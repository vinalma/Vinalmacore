# Skill: Landing Pages

## Quando usar
- Criar ou revisar landing pages de captura, vendas ou agendamento

## Inputs mínimos
- Objetivo (lead, venda, agendamento)
- Produto/oferta
- Público
- CTA principal

## Estrutura de LP de captura

```
1. Hero          → Headline + sub + CTA
2. Dor/Desejo    → Problema que resolve
3. Solução       → O que é + como funciona
4. Prova         → Depoimentos, números, logos
5. Oferta        → O que inclui + urgência/escassez
6. CTA final     → Único e claro
```

## Checklist técnico

- [ ] Velocidade < 3s (PageSpeed > 80)
- [ ] Mobile responsivo
- [ ] Pixel / tag instalado
- [ ] Formulário conectado ao CRM ou planilha
- [ ] Evento de conversão ativo
- [ ] Domínio próprio (sem `/linktree` ou similar)

## Headline fórmulas

```
[Verbo de resultado] + [benefício] + [sem/mesmo que objeção]
Ex: Consiga mais leads sem aumentar o orçamento de tráfego

[Número] + [resultado] + [tempo]
Ex: 3x mais agendamentos em 30 dias
```
