# Skill: Automação Básica

## Quando usar
- Automatizar captura de leads, notificações, follow-up, relatórios

## Inputs mínimos
- Gatilho (o que dispara a automação)
- Ação (o que acontece)
- Ferramentas disponíveis

## Automações mais usadas

| Gatilho | Ação | Ferramenta |
|---|---|---|
| Lead gerado | WhatsApp + email | Make / Zapier |
| Formulário enviado | Planilha + notificação | Make |
| Pagamento aprovado | Acesso + email boas-vindas | Hotmart / Kiwify |
| Sem resposta em Xh | Follow-up automático | ManyChat / CRM |

## Template de mapeamento

```
Gatilho: [o que inicia]
Condição: [filtro, se necessário]
Ação 1: [primeira ação]
Ação 2: [segunda ação, se necessário]
Ferramenta: [Make / Zapier / n8n / nativo]
```

## Prioridade de automação

1. Captura e registro de lead
2. Notificação imediata (você + lead)
3. Follow-up se não responder
4. Relatório semanal automático

## Regra

> Automatize o que você faz manualmente toda semana.
> Não automatize o que ainda não funciona.
