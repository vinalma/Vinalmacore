# Skill: Ofertas

## Quando usar
- Criar, estruturar ou revisar uma oferta

## Inputs mínimos
- Produto/serviço
- Público
- Ticket (ou objetivo de preço)
- Contexto (lançamento, campanha, evergreen)

## Elementos de uma boa oferta

```
1. Resultado claro       → O que a pessoa conquista
2. Prazo                 → Em quanto tempo
3. Diferenciador         → Por que com você
4. Bônus (opcional)      → Aumenta valor percebido
5. Garantia              → Reduz risco
6. Urgência/Escassez     → Motivo para agir agora
7. Preço + CTA           → Claro e único
```

## Fórmula de posicionamento

```
Eu ajudo [público] a [resultado] sem [objeção principal]
mesmo que [crença limitante]
```

## Oferta de entrada (front-end)
- Preço baixo ou gratuito
- Entrega rápida e tangível
- Porta para a oferta principal

## Stack de oferta

```
Lead magnet (grátis)
  → Oferta de entrada (R$ 47–197)
      → Oferta principal (R$ 497–2k)
          → Continuidade / upsell
```
