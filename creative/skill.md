# Skill: Criativos

## Quando usar
- Briefar criativos para anúncios, stories, feed, UGC

## Inputs mínimos
- Formato (feed, story, reels, banner)
- Objetivo (parar scroll, gerar clique, gerar lead)
- Produto/serviço
- Tom visual (aspiracional, direto, emocional)

## Estrutura de criativo de anúncio

```
[0-3s]  Hook visual — para o scroll
[3-8s]  Problema ou desejo
[8-15s] Solução + diferenciador
[15s+]  CTA direto
```

## Brief rápido

```
Formato: 
Objetivo: 
Produto: 
Público: 
Hook (frase de abertura): 
Visual principal: 
CTA: 
```

## Boas práticas

- Thumb para o scroll: rosto, texto grande ou contraste forte
- Hook nos primeiros 3 segundos
- CTA explícito e único
- Mobile first (vertical 9:16)
- Teste versão sem áudio (legenda ou texto em tela)
